# MCP Memory Server

Servidor MCP de memoria persistente construido con Memento.

## Requisitos

- Node.js 18+
- Docker y Docker Compose

## Instalación

```bash
npm install
```

## Uso con Docker

```bash
docker-compose up --build
```

## Uso local

```bash
./start
```

## Configuración

La base de datos se guarda en `/data/memory.db` dentro del contenedor.
